Project: Local Cricket Live (Flutter + Django + Realtime)

I built a Cricbuzz-style local cricket live scoring platform with realtime updates:
- Flexible match setup (custom overs and players/team)
- Ball-by-ball scoring with extras + wickets
- Live Summary, Scorecard, Commentary tabs
- WebSocket-based realtime sync (no manual refresh)
- JWT-protected admin scoring flow
- Player career stats (batting + bowling aggregates)

Tech:
Flutter, Django REST Framework, Django Channels, PostgreSQL, Redis, JWT (SimpleJWT), Railway

If anyone wants a walkthrough/demo, I can share it.
